

# Tech stack

- Express 4
- cross-env
- Babel
- Webpack 3


# Scripts

## Build the Project
Create the bundle with Webpack

    $ npm run dev:build

    $ npm run prod:build

## Start the Service

    $ npm run dev

    $ npm run prod
